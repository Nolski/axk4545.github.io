FAQ

  • I just finished the installation, now what?

    • Congratulations! Explore the sections of the Welcome Screen dialog. Get used to the desktop. If your system uses special drivers such as those for an AMD or Nvidia graphics card, you probably want to start with the Drivers section. The New Features section may also be of interest.

  • Help! I am stuck and I don't know how to do X.

    • If you need support with Linux look to the community. The Linux Mint forums can be found here You can alos get support through IRC by going here and setting the channel to #linuxmint-help

  • How do I learn more about command Y?

    • Linux has many ways to view information about commands on the system. One of the most common is man. To use it open a terminal and type man [command] where command is the command you want to learn about (man pages can also be viewed online at linux.die.net/man). You can also use the info utility. It has the same syntax as man.

  • What is a root user and why is it special?

    • The root user on a Linux or UNIX® system is the administrative user and has full, unrestricted access to the system. As such you should set a strong root password. The sudo command can be used to temporarily elevate a users privileges to that of root. To learn more about the sudo command see http://linux.die.net/man/8/sudo
